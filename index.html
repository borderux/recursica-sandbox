Hex Color PR Creator

Enter a hex color value and press Enter to create a pull request.

No account required - anyone can contribute!

How it works:

  1. Enter a hex color and press Enter
  2. Click the "Create File" button that appears
  3. GitHub will open with a pre-filled file creation page
  4. Submit the file - GitHub automatically creates a PR!

Super simple! Uses GitHub's built-in file creation workflow.
