Hex Color PR Creator

Enter a hex color value and press Enter to create a pull request.

The color will be saved to a file in the repository.

Setup Required

To use this app, you need to:

  1. Create a GitHub Personal Access Token with repo permissions
  2. Add it to the GITHUB_TOKEN constant in the JavaScript
  3. Or use GitHub's built-in authentication by setting up a GitHub OAuth App

Note: For security, consider using GitHub Actions or a backend service to handle authentication.
